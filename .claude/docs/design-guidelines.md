# DevLaunch デザインガイドライン

## デザイン方針

macOS ネイティブ準拠。システム標準のカラー・フォント・コンポーネントを使用し、カスタムスタイルは最小限に抑える。

## カラー

| 用途 | カラー | 備考 |
|------|--------|------|
| 背景 | システム標準（`.background`） | ポップオーバー・設定画面 |
| テキスト | システム標準（`.primary`, `.secondary`） | |
| アクセント | `.accentColor`（システム設定に追従） | ボタン・選択状態 |
| エラー | `.red` | コマンド未検出時のエラー表示 |

## タイポグラフィ

| 用途 | フォント |
|------|---------|
| プロジェクト名 | `.body` |
| 設定ラベル | `.body` |
| エラーメッセージ | `.caption` + `.secondary` |
| セクションヘッダー | `.headline` |

## コンポーネント

- **ポップオーバー**: `NSPopover` / SwiftUI `.popover` 標準
- **リスト**: SwiftUI `List` 標準スタイル
- **設定画面**: SwiftUI `Form` + `Settings` シーン
- **ボタン**: システム標準スタイル

## レイアウト

- **ポップオーバー幅**: 300pt
- **ポップオーバー最大高**: 400pt（スクロール対応）
- **リスト行高**: 36pt
- **余白**: システム標準のパディング
